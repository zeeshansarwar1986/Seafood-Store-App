🇬🇧 Project Goals

Designed to provide a high-end shopping experience for seafood lovers. Features persistent cart, real-time price updates, and a smooth checkout flow.

⚓ Technology Stack

Built using Flutter and Dart. State management via Provider. Sophisticated UI using Google Fonts, Animate Do, and Carousel Slider.

🛒 Commercial Features

⚖️
Dynamic Pricing: Automatic price adjustment based on weight selections.
persistent
Safe Cart: Shopping cart items stay saved even if the app is closed.
💳
Streamlined Checkout: Fast 3-step process to complete orders.

✨ Design Excellence

🌊
Ocean UI: Themed glassmorphism aesthetic tailored for seafood.

🛠️ Development Tools

  • Provider: Centralized state management for performance.
  • Dio: Robust HTTP networking for future API integration.
  • Shared Preferences: Local storage for cart persistence.
  • Animations: Animate Do & Staggered Animations for premium feel.

📝 How to Customize

Modify lib/services/mock_api_service.dart to change the products list. Update lib/theme/app_theme.dart to change the primary ocean colors.
